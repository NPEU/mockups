Badge

The badge is used for any image that represents a brand, identity, accolage, status or similar marking.

Every effort should be made to provide a badge as an SVG. The build process will generate a fallback PNG. So that this PNG is of a resonable quality, SVG's should be around 300px high and include width and heighty attributes.

SVGO removes these attributes for the final SVG output, but svg2png requires that they're present intitially.
