A note on Blockrow

Currently, l-bloackrow does not apply any styles, so it may seem like a redundant container. However, it's value is in it's delimitiing effect; it makes it clear that the contents of the element where this class is applied is intended to take a full-width of the screen, and never be changed due to responsiveness.

All proportions are also available at "wide" breakpoint (default 45em).
